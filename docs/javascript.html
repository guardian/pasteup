Micro libraries — AMD compatible

There are three recommended ways to use these libraries.

  • Link via a script element and use as a normal standalone JavaScript library.
  • Asynchronously downloaded by any AMD compatible script loader, eg. require.js or curl.js
  • Synchronously downloaded and included in a project's AMD compatible build process.
