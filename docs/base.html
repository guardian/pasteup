Base CSS

Headings

Six heading styles (potentially more) are defined which can be used on any heading element by use of the correct class value.

.article-heading - 24px/30px

.sub-heading - 18px/30px

.related-heading - 16px/20px

.trailblock-heading - 14px/20px

.article-base-heading - 15px/20px
.base-heading - 12px/15px

Various specialised heading styles are also available.

Layout helpers

  • .pull-right - Adds a float:right value to the element.
  • .pull-left - Adds a float:left value to the element.
  • .margin-top - Adds a margin-top value of 15px, the default line-height.
  • .margin-bottom - Adds a margin-bottom value of 15px, the default line-height.
  • .cf - Adds a clearfix to the element. Specifically, the Micro clearfix.
